Either attach a one-time event listener to this Component or attach an observed event listener to another Component or Element. It will be called a maximum of once, then removed. Event listeners on Components also receive DOM events from the Component's Element. If the provider argument is provided, the listener is attached to the Element or Component's Element and automatically cleaned up when this Component is disposed.
