spare .videojs.Component#enableTouchActivity~summary

Enable reporting user touch activity when any touch events occur. This is normally disabled because it is not typically desirable for the touchend event to appear to be user activity. For example, in the Player Component asynchronous handling of touch activity events can cause the video controls to immediately reappear after being manually dismissed.
