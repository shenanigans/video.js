Get or set the styled width of the Component.

Arguments

optional
| Number | String width

Set the width to a Number or String. Note that Strings are used literally and should end in px or %.
optional
| Boolean skipListeners

Do not trigger a resize Event.

Return Values

| videojs.Component | Number

When setting the width argument, this Component is returned. Otherwise the current width is returned as a Number of pixels.
